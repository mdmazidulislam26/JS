# Set up backend project
---
- initialize project name and description...
- ```javascript
  npm init
  ```
- connect github
- change branch
- ```javascript
   git branch -M main
  ```
- set remote
- ```javascript
   git remote add origin https://github.com/xyz
  ```
- set upstream
- ```javascript
  git push -u origin main
  ```
